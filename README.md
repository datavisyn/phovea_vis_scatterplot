=====================

phovea datavisyn-scatterplot binding

Installation
------------

```
git clone https://github.com/datavisyn/phovea_vis_scatterplot.git
cd phovea_vis_scatterplot
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
